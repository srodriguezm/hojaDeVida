Software Architecture Project

Software Architecture Project

Description:

NIDOO is an StartUp trying to connect parking lot owners with car ussers in order to make money based in the short-therm market. They asked for some Consultancy with the university of Los Andes, which assigned that tasked to the undergraduate Software Architecture students (2018-2) in order to improve their current situation.

We proposed a lambda based architecture in order to improve latency, scalability and availability. This part of the architecture was developed at the back-end with Flask in Python, There was as well a Kafka server wich played the role of charge balancer to improve the performance attributes. In the other hand, the front-end was developed with node-js.
